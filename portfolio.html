Applicazioni Desktop

Gestore Turni

In Evidenza

Applicazione desktop in Electron.js per la gestione automatica dei turni lavorativi. Permette di creare ruoli, turni, dipendenti e gruppi, e con un click genera un planning settimanale ottimizzato che rispetta tutti i vincoli contrattuali e legali (riposo 11h, ore massime, ecc.).

Caratteristiche principali:

  • Privacy totale: Database SQLite cifrato AES-256, funzionamento offline
  • Algoritmo intelligente: Genera svariate varianti e seleziona la migliore
  • Rotazione automatica: Sistema che alterna i turni tra i dipendenti
  • Modifica manuale: Planning editabile con verifica vincoli real-time
  • Gestione vincoli: Rispetta ore contrattuali, riposi, indisponibilità, gruppi
  • Esportazione: Planning esportabile su richiesta dell'utente

Progressive Web Apps (PWA)

Come Installare le PWA

Le Progressive Web Apps possono essere installate come applicazioni native sul tuo dispositivo, funzionando anche offline dopo l'installazione.

Per PC o Android

Installa come Progressive Web App (PWA) per una gestione sicura anche offline sul tuo PC o dispositivo Android.

  • Apri nel Browser: Apri la pagina in un browser supportato (es. Chrome, Edge, Samsung Internet) su HTTPS
  • Installa l'App: Clicca sul pulsante di installazione e successivamente premi installa
  • Accedi all'App: Gestisci i tuoi dati in modo sicuro, anche offline

Per iOS

  • Premi i Tre Puntini: Tocca i tre puntini accanto alla barra di ricerca in Safari
  • Seleziona Condividi: Tocca Condividi dal menu
  • Tocca Altro: Seleziona Altro (tre puntini) dalle opzioni di condivisione
  • Aggiungi alla Schermata Home: Seleziona Aggiungi alla Schermata Home (quadrato con un + al centro)
  • Abilita Web App: Spunta Apri come Web App

Nota: Questa funzione non scarica il sito ma permette di visualizzarlo come app. Per usare offline, apri la scorciatoia, carica la pagina e poi togli la connessione.

Progetti di Stage e Scolastici

Stage Scolastico – Sviluppo Applicazioni e Web App

Stage: 2024

Durante lo stage scolastico ho progettato e sviluppato due applicazioni complete, occupandomi sia della logica di backend sia dell'organizzazione dei dati, con particolare attenzione all'automazione dei processi e all'usabilità.

Gestione delle Richieste Clienti da File CSV

Ho sviluppato in autonomia un'applicazione in Python con framework Flask per la gestione delle richieste ricevute quotidianamente dal sito aziendale. Il sistema elaborava automaticamente tre file CSV generati a fine giornata e un file TXT contenente i numeri di telefono dei clienti già registrati. Le richieste venivano classificate in modo automatico in tre categorie: Da contattare, Clienti già noti, Numeri di telefono non validi. I risultati venivano infine esportati in un file XLSX strutturato con tre colonne dedicate, semplificando il lavoro del reparto commerciale. Il progetto è stato completato con successo in due giorni.

Web App per la Gestione delle Ordinazioni di Bevande

Ho realizzato una web app completa per la gestione delle ordinazioni di bevande aziendali, con sistema di autenticazione e ruoli differenziati (amministratore e utente). Ogni bevanda era associata a una disponibilità aggiornata in tempo reale tramite un database relazionale, garantendo un controllo accurato delle scorte. Al termine di ogni settimana, l'applicazione generava automaticamente un report riepilogativo che mostrava, per ciascun dipendente, il numero di bevande ordinate, facilitando il monitoraggio dei consumi interni.

Sito di Ricette con ASP.NET Minimal API

Progetto conclusivo

Progetto di sito web dedicato alla cucina sviluppato con un back-end basato su ASP.NET Minimal API e un front-end responsive realizzato con HTML5, CSS3, JavaScript e Bootstrap.

L'applicazione integra un database MariaDB per la gestione delle ricette e delle recensioni degli utenti e utilizza l'API esterna TheMealDB per consentire la ricerca delle ricette in base alla regione di origine.

Funzionalità principali

  • Visualizzazione, creazione, modifica ed eliminazione di ricette e recensioni
  • Implementazione di endpoint RESTful (GET, POST, PUT, DELETE)
  • Interfaccia utente dinamica con hamburger menu e card responsive
  • Ricerca di ricette per regione tramite integrazione API esterna

Il progetto è stato realizzato con particolare attenzione all'integrazione di servizi esterni, alla struttura delle API e al design responsivo, garantendo un'esperienza utente intuitiva su dispositivi desktop e mobile.
